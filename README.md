# 🤖 AI Automation Projects

This repository contains automation projects built using APIs, AI, and workflow tools to improve productivity and efficiency.

---

## 🚀 Projects

### 📧 AI Gmail Auto Sorter

* Automatically classifies emails into categories like spam, useful, and sponsored.
* Filters important emails for entrepreneurs.
* Extracts useful emails and stores them in Excel for easy tracking.

---

### 🤖 Telegram ChatGPT Bot

* Integrated ChatGPT with Telegram using API.
* Allows users to interact with AI directly inside Telegram.
* Supports real-time responses.

---

### ⏰ Email Reminder System

* Sends automated email reminders.
* Useful for task tracking and notifications.

---

### 🌦️ Weather Report System

* Fetches real-time weather data using APIs.
* Provides weather updates based on user input.

---

## 🛠️ Tech Stack

* Python
* APIs (OpenAI, Telegram, Email, Weather)
* Automation Tools (n8n / custom scripts)

---

## 🎯 Purpose

These projects demonstrate practical use of AI and automation to solve real-world problems and improve productivity.
