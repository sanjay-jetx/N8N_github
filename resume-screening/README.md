# 📄 Resume Screening AI (n8n Workflow)

## 🚀 Overview
This project automates the resume screening process using AI and n8n. 
The system helps HR teams quickly analyze candidates based on job requirements.

## ⚙️ How It Works
1. HR provides the required qualifications and skills.
2. The AI agent reads candidate resumes.
3. It compares the resume with the required skills.
4. The system analyzes:
   - Skills match
   - Experience relevance
   - Strengths
   - Weaknesses
5. The results are stored in an Excel sheet.

## 📊 Output (Excel Data)
The system automatically generates structured data including:
- Candidate Name
- Email
- Phone Number
- Skills
- Strengths
- Weaknesses
- Match Rating (Score)

## 🛠️ Tech Stack
- n8n (Workflow Automation)
- AI Agent (OpenAI / LLM)
- Excel / Google Sheets
- JSON-based workflow

## 🎯 Use Case
- Automates resume filtering
- Saves HR time
- Improves candidate shortlisting accuracy
